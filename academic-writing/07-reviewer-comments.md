# 审稿人回复模板库

## 一、Response Letter结构

```
Dear Editor,

Thank you for the opportunity to revise our manuscript [ID: XXX].
We have carefully addressed all comments from the reviewers.
Below we provide point-by-point responses.

[修改总结 - 2-3句话概括主要修改]

Sincerely,
[Authors]
```

### 完整模板

```
Response to Reviewer Comments
Manuscript ID: [XXX]
Title: [Title]

================================================================
REVIEWER 1
================================================================

Comment 1.1: [审稿人原文逐字引用]

Response: Thank you for this insightful comment. [回复内容].
[说明具体修改，标注修改位置] (see Page X, Lines X-X, highlighted
in blue)

Comment 1.2: [审稿人原文]

Response: [回复]

================================================================
REVIEWER 2
================================================================

...
```

## 二、回复句式库

### 2.1 同意并修改

```
英文：
- "We thank the reviewer for this valuable suggestion. We have
   revised the text accordingly (Page X, Lines X-X)."
- "We agree with the reviewer and have added [具体内容] as
   suggested (Page X)."
- "This is an excellent point. We have incorporated the suggested
   analysis/clarification."

中文：
- "感谢审稿人的建议。我们已按要求修改（见第X页，第X-X行）。"
- "我们同意审稿人的观点，已补充说明如下：..."
```

### 2.2 同意但部分采纳

```
- "We appreciate this suggestion. While we agree that [部分同意],
   we have chosen to retain [保留部分] because [理由]. However,
   we have added a clarification in the Discussion (Page X)."
- "Thank you for this comment. We have addressed it by [具体做法],
   which we believe achieves the same purpose as suggested."
```

### 2.3 礼貌地不同意

```
- "We respectfully disagree with this interpretation for the
   following reasons: (1)... (2)... However, we have revised the
   text to better explain our reasoning (Page X)."
- "We thank the reviewer for raising this point. While [审稿人观点],
   our data suggest [我们的证据]. We have added a discussion of
   this alternative interpretation (Page X)."
- "This is a valid concern. However, [解释原因]. We have added a
   sentence acknowledging this limitation (Page X)."
```

### 2.4 需要补充实验/数据

```
- "We appreciate the reviewer's suggestion to [实验建议].
   Due to [限制原因], we were unable to perform this analysis.
   However, we have [替代做法] to address this concern."
- "We agree this would strengthen the study. We have added [补充内容]
   in the revised manuscript. [说明结果]"
```

### 2.5 审稿人误解/不清晰

```
- "We apologize for the unclear description. We have revised the
   text to better explain [内容] (Page X, Lines X-X)."
- "We thank the reviewer for pointing out this ambiguity. We have
   rewritten this section to clarify that [澄清内容]."
```

## 三、各类型审稿意见回复模板

### 3.1 "The novelty is insufficient"

```
Response: We thank the reviewer for this concern. In the revised
manuscript, we have:
(1) Strengthened the novelty statement in the Introduction (Page X),
    clearly articulating how this study differs from previous work.
(2) Added a paragraph in the Discussion highlighting the unique
    contributions (Page X).
(3) Revised the Abstract to emphasize the novel aspects.

Specifically, this study is the first to [具体创新点], which
distinguishes it from prior studies that [对比].
```

### 3.2 "The Discussion needs improvement"

```
Response: We agree with the reviewer that the Discussion needed
more depth. We have substantially revised the Discussion section:

(1) Added mechanistic explanations for [发现] (Page X, Lines X-X).
(2) Included comparison with [补充文献] studies.
(3) Expanded the discussion of [具体方面].

The revised Discussion now covers [概括改进内容].
```

### 3.3 "More references are needed"

```
Response: We have added [N] additional references to strengthen the
literature support:
- [Ref X]: Supports the statement about [内容] (Page X)
- [Ref X]: Provides comparison data for [内容] (Page X)
- [Ref X]: Discusses [相关内容] (Page X)

The total number of references has increased from [N1] to [N2].
```

### 3.4 "Statistical analysis needs improvement"

```
Response: We thank the reviewer for this important suggestion. We
have improved the statistical analysis:

(1) Added [检验方法] to verify [内容].
(2) Provided confidence intervals for all regression parameters.
(3) Added a multiple comparison correction (Bonferroni/Holm) for
    [分析].

The revised results are presented in Table X and discussed on Page X.
```

### 3.5 "The figures need improvement"

```
Response: We have revised the figures as suggested:
- Fig. X: [具体修改内容]
- Fig. X: [具体修改内容]
- Added new Fig. X to show [新内容]

All figures now meet the journal's resolution and formatting
requirements.
```

### 3.6 "English language needs improvement"

```
Response: We thank the reviewer for noting the language issues.
The manuscript has been thoroughly revised by a professional
editing service [or: a native English-speaking co-author].
Changes are too extensive to mark individually, but the entire
manuscript has been improved for clarity and readability.
```

## 四、Rebuttal策略

### 4.1 核心原则

1. **永远先感谢** — 不管意见多尖锐
2. **引用原文** — 逐字引用审稿人的话
3. **有理有据** — 用数据/文献支撑回复
4. **标注位置** — 每次回复都标注修改位置
5. **区分轻重** — 重要问题详细回复，小问题简洁回复

### 4.2 回复长度指南

| 意见类型 | 回复长度 |
|---------|---------|
| 简单格式问题 | 1-2句 |
| 需要解释的观点 | 1段（3-5句） |
| 需要补充数据 | 2-3段 + 图/表 |
| 核心争议点 | 详细回复 + 文献支撑 |

### 4.3 常见陷阱

- **不要反驳情绪** — 只回应事实
- **不要过度道歉** — "We apologize"用一次就够
- **不要空口承诺** — "Future work will..."不如具体修改
- **不要遗漏任何意见** — 逐条回应，包括minor comments
- **不要大幅重写** — 修改范围要与意见程度匹配

## 五、审稿回复模板（环境科学版）

### 针对"Discussion不足"的回复

```
Comment: The Discussion section lacks mechanistic explanations and
is largely a repetition of the results.

Response: We sincerely thank the reviewer for this constructive
criticism. We have completely rewritten the Discussion section
with the following improvements:

(1) Mechanistic explanations: Each finding is now accompanied by
    a biogeochemical mechanism. For example, we now explain why
    CH4 concentrations increase along the pipeline in terms of
    progressive DO depletion and methanogenic activity (Page X,
    Lines X-X).

(2) Literature comparison: We have added comparisons with X
    published studies, including [作者] [年份] who reported
    similar findings in [地点].

(3) Integration: We have added a new subsection (Section X.X)
    discussing the implications of our findings for carbon
    emission accounting in urban sewage systems.

The revised Discussion is approximately [X] words, compared to
[X] words in the original version.
```

### 针对"样本量不足"的回复

```
Comment: The sample size (n=X) may not be sufficient to support
the statistical conclusions.

Response: We appreciate this concern. While our sample size is
relatively modest, we have taken the following steps to ensure
statistical robustness:

(1) We performed power analysis and confirmed that our sample
    size provides >80% power to detect the observed effect sizes
    at α=0.05.

(2) We supplemented parametric tests with non-parametric
    alternatives (Mann-Whitney U), which do not assume normality.

(3) We have added a statement acknowledging the sample size
    limitation in the revised Discussion (Page X).

We note that similar studies in the field [references] have used
comparable sample sizes.
```
