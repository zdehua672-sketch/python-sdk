# 中文论文规则库

## 一、中文核心论文格式规范

### 1.1 与SCI论文的关键区别

| 维度 | 中文核心 | SCI英文 |
|------|---------|---------|
| 语言 | 中文为主，术语附英文 | 英文 |
| 摘要 | 结构化（目的/方法/结果/结论），300-500字 | 结构化，250词 |
| 关键词 | 3-8个，中英文对照 | 3-6个，英文 |
| 参考文献 | GB/T 7714格式 | APA/期刊自定义 |
| 图表标题 | 中文标题+英文标题 | 英文标题 |
| 字数 | 正文8000-15000字 | 4000-8000词 |
| 单位 | 国际单位制，中文叙述 | SI单位 |

### 1.2 GB/T 7714参考文献格式

**期刊论文：**
```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```
示例：
```
[1] 张三, 李四, 王五. 城市污水管网碳污染物分布特征研究[J]. 环境科学, 2023, 44(3): 1234-1242.
```

**英文期刊：**
```
[序号] AUTHOR A, AUTHOR B. Title[J]. Journal, Year, Vol(No): Pages.
```
示例：
```
[2] GUISOSONA A, EASTER C, GUTIERREZ O. Estimation of wastewater intr residence time in sewers[J]. Water Science and Technology, 2008, 57(10): 1601-1606.
```

**学位论文：**
```
[序号] 作者. 题名[D]. 保存地: 保存单位, 年份.
```

**专著：**
```
[序号] 作者. 书名[M]. 出版地: 出版社, 年份: 页码.
```

### 1.3 稿件排版要求

```
标题：二号黑体，居中
作者：四号楷体，居中
单位：小五号宋体，居中
摘要：小五号宋体，"摘要"二字用小五号黑体
关键词：小五号宋体，"关键词"三字用小五号黑体
正文：五号宋体
一级标题：四号黑体
二级标题：五号黑体
三级标题：五号楷体
图表标题：小五号黑体
参考文献：小五号宋体
```

## 二、中文学术语言特点

### 2.1 用词规范

**避免口语化：**
| 口语 | 学术用语 |
|------|---------|
| 很多 | 显著/大量/较高 |
| 差不多 | 近似/约/大约 |
| 好 | 优良/良好/较好 |
| 坏 | 较差/偏低/不足 |
| 搞清楚 | 阐明/揭示/探究 |
| 用了 | 采用/运用/利用 |
| 一般来说 | 通常/一般而言/总体而言 |
| 跟...有关 | 与...呈显著相关/密切相关 |

**连接词使用：**
```
递进：此外、进一步、同时、与此同时
转折：然而、但、不过、尽管如此
因果：因此、因而、由此、鉴于此
总结：综上所述、总体而言、由此可见
对比：相比之下、不同于、与...相比
```

### 2.2 标点规范

- **中文标点**：全角（，。；：""）
- **数字和英文**：半角，前后不加空格
- **括号**：中文叙述用全角（），公式/代码/文献编号用半角()
- **百分号**：数字与%之间不留空格，如"85.3%"
- **单位**：数字与单位之间留一个半角空格，如"10 mg/L"

### 2.3 数字与单位

- **正文叙述**：一般用阿拉伯数字
- **概数**：用中文数字，如"约百分之八十"
- **单位格式**：`数值 + 空格 + 单位`，如 `25.3 ± 1.2 mg/L`
- **万/亿**：中文论文可使用万、亿，如"3.5万m³/d"

## 三、各章节写作要点

### 3.1 摘要

**结构：**
```
[目的] 为了解决...问题，本文...
[方法] 采用...方法，对...进行了研究。
[结果] 结果表明：(1)...；(2)...；(3)...。
[结论] 研究表明...，为...提供了科学依据。
```

**字数要求：** 300-500字（中文核心通常300字左右）

**示例（本课题）：**
```
[目的] 为揭示校园污水管网中碳污染物在固-液-气三相中的赋存特征
及碳平衡规律，本研究以某高校校园污水管网为对象。[方法] 通过布设
监测点采集不同功能区、不同季节的污水、沉积物和气体样品，测定总
有机碳(TOC)、化学需氧量(COD)、甲烷(CH4)、二氧化碳(CO2)等指标，
结合主成分分析(PCA)和层次聚类分析(HCA)探讨碳污染物的空间分异
特征。[结果] 结果表明：(1)餐饮区TOC浓度显著高于教学区和生活区
(p<0.05)；(2)管网末端CH4浓度较管口升高了2-3倍；(3)冬季CH4浓度
与DO呈显著负相关(r=-0.82, p<0.01)。[结论] 研究表明，管网功能区
类型和沿程厌氧程度是影响碳污染物多相态分布的关键因素，本研究为
校园污水管网碳排放核算提供了基础数据。
```

### 3.2 引言

**逻辑链：**
```
宏观背景 → 领域现状 → 具体问题 → 文献综述 → 研究空白 → 研究目标
```

**中文引言的"倒金字塔"：**
1. **第一段（100-150字）：** 宏观背景
   - "随着城市化进程加快，..."
   - "碳达峰碳中和目标下，..."
2. **第二段（150-200字）：** 领域现状
   - "目前，国内外学者对...开展了大量研究"
   - "XXX等[1-3]研究发现..."
3. **第三段（100-150字）：** 研究空白
   - "然而，现有研究存在以下不足：(1)...；(2)...；(3)..."
4. **第四段（50-100字）：** 研究目标
   - "基于此，本研究以...为对象，采用...方法，..."

### 3.3 材料与方法

**结构：**
```
2.1 研究区域概况
2.2 样品采集与分析方法
2.3 数据处理与统计分析
```

**要点：**
- 明确采样时间、地点、频率
- 注明分析方法的国标编号（HJ/GB）
- 统计软件和方法需要交代版本

**示例（本课题）：**
```
2.2.1 水样采集
采用有机玻璃采水器在各监测点采集表层水样，采集后立即用0.45μm
滤膜过滤，滤液用于溶解性有机碳(DOC)测定，未过滤水样用于总有机
碳(TOC)测定。水样保存于4°C棕色玻璃瓶中，24h内完成分析。
```

### 3.4 结果

**组织原则：** 先整体后局部，先一般后特殊

**结构建议：**
```
3.1 碳污染物浓度总体特征（描述统计）
3.2 空间分布特征（不同功能区对比）
3.3 季节变化特征（冬季vs春季）
3.4 相关性分析结果
3.5 主成分分析结果
```

**数据→文字的核心句式：**
```
"[地点/条件]的[指标]为[X±Y] mg/L，[高于/低于][对比对象]"
"[指标A]与[指标B]呈[显著/极显著][正/负]相关(r=X, p<0.05)"
"PCA结果表明，前[N]个主成分解释了[X]%的方差"
```

### 3.5 讨论

**中文Discussion的核心要求：**
1. **不是重复结果**，而是解释结果
2. **每个发现都要有机制解释**
3. **要与文献对比**
4. **要指出创新点**

**结构模板：**
```
4.1 [主要发现1]的机制解释
    → 本研究发现...
    → 从微生物代谢角度解释...
    → 文献对比：与XX等[ref]的研究一致/不同

4.2 [主要发现2]的机制解释
    → 同上逻辑

4.3 [主要发现3]的机制解释
    → 同上逻辑

4.4 研究意义与应用价值

4.5 研究局限性
```

### 3.6 结论

**要求：**
- 3-5条，每条1-2句
- 不引用文献
- 不用"本文"开头
- 要有明确的研究贡献

**模板：**
```
(1) [发现]...，[意义]...
(2) [发现]...，[意义]...
(3) [发现]...，[意义]...
```

## 四、中文论文常见格式问题

### 4.1 摘要问题
- 摘要中出现参考文献编号 `[X]`
- 摘要中使用"本文""本论文"
- 结论部分没有定量结果

### 4.2 引言问题
- 引言过短（<500字），缺乏文献支撑
- 没有明确的研究空白陈述
- 引用文献过旧（>10年）

### 4.3 方法问题
- 未注明方法来源（国标/文献编号）
- 统计方法描述不完整
- 缺少质控措施说明

### 4.4 结果问题
- 图表重复展示同一数据
- 结果与讨论混在一起
- 缺少统计检验的p值

### 4.5 讨论问题
- Discussion变成"结果的复述"
- 缺少机制解释（最常见问题）
- 文献对比只说"一致"不说原因

### 4.6 结论问题
- 结论重复摘要内容
- 结论中出现参考文献
- 过度夸大研究意义

## 五、本课题写作示例

### 示例：Discussion段落（机制解释+文献对比）

```
4.1 管网碳污染物空间分异机制

本研究发现，餐饮区TOC浓度(185.6±42.3 mg/L)显著高于教学区
(98.4±28.7 mg/L)和生活区(112.5±31.2 mg/L)(p<0.05)。这一现象
可以从有机物来源角度解释：餐饮区排放的废水含有大量油脂和食物残渣，
其有机负荷远高于教学区实验室废水和生活区生活污水[1-3]。Guisasola
等[4]研究表明，餐饮废水的BOD/COD比值通常在0.5-0.7之间，表明其
有机物可生化性较强，有利于微生物快速利用。

沿程方向上，管网末端DO浓度(0.3±0.2 mg/L)显著低于管口
(2.8±0.6 mg/L)(p<0.01)，同时CH4浓度由管口的0.8±0.3 mg/L升高
至末端的3.2±1.1 mg/L。这一沿程变化规律与Sun等[5]的观测结果一致。
从微生物代谢角度来看，沿程有机物的好氧降解逐步消耗水体中的溶解氧，
当DO降至0.5 mg/L以下时，厌氧产甲烷菌活性显著增强[6]，从而导致
CH4浓度的累积性升高。
```

### 示例：引言首段

```
随着"碳达峰、碳中和"战略目标的提出，城市碳排放核算与减排成为
环境科学研究的热点领域。城市污水管网作为连接居民生活与污水处理厂
的重要基础设施，不仅是污染物输运的通道，更是碳污染物发生物理、
化学和生物转化的重要场所[1-3]。研究表明，污水管网系统每年产生的
甲烷(CH4)排放量约占全球人为CH4排放总量的5%-10%[4]，其碳排放
贡献不容忽视。
```
