# 常见错误库

## 一、标题（Title）常见错误

| 错误类型 | 错误示例 | 正确写法 |
|---------|---------|---------|
| 太宽泛 | "碳污染物研究" | "校园污水管网固-液-气多相态碳污染物赋存特征" |
| 缩写未展开 | "TOC分布特征" | "总有机碳(TOC)分布特征" |
| 包含结论 | "CH4显著升高" | "CH4沿程变化特征" |
| 中英文混用 | "基于PCA的碳污染物analysis" | "基于主成分分析的碳污染物分布特征研究" |
| 太长 | >25字（中文）/>15词（英文） | 精简到核心要素 |

---

## 二、摘要（Abstract）常见错误

### 中文摘要

| 错误 | 说明 | 修正 |
|------|------|------|
| 包含参考文献编号 | "研究表明[1]..." | 删除引用 |
| 包含图表引用 | "如图1所示" | 直接描述数据 |
| 使用"本文" | "本文研究了..." | "研究了..." |
| 缺少定量结果 | "浓度较高" | "浓度为185.6±42.3 mg/L" |
| 结论空洞 | "为相关研究提供了参考" | "为管网碳排放核算提供了基础数据" |

### 英文摘要

| 错误 | 说明 |
|------|------|
| 中式英语 | "The result showed that..." → "The results indicate that..." |
| 时态混乱 | Methods用现在时（应为过去时） |
| 主谓不一致 | "The data shows" → "The data show" |

## 三、引言（Introduction）常见错误

### 3.1 逻辑问题

```
错误：直接切入技术细节，没有背景铺垫
正确：宏观背景 → 领域现状 → 具体问题 → 文献 → 空白 → 目标

错误：文献综述变成罗列
正确："张三[1]和李四[2]发现...，但王五[3]的研究表明..." 

错误：没有明确的研究空白
正确："然而，现有研究存在以下不足：(1)...(2)...(3)..."
```

### 3.2 引用问题

| 错误 | 说明 |
|------|------|
| 引用过旧 | 引用文献>50%为10年前的 |
| 自引过多 | 自引文献>总引用的30% |
| 引用不准确 | 引用内容与原文不符 |
| 引用范围窄 | 只引中文或只引英文 |
| 二手引用 | 引用A论文中提到的B结论（应找原文B） |

## 四、材料与方法（Methods）常见错误

| 错误 | 说明 | 修正 |
|------|------|------|
| 方法来源不明 | "采用分光光度法测定COD" | "采用重铬酸钾法(HJ 828-2017)测定COD" |
| 缺少质控 | 无质控说明 | "每批样品设置3个平行样，加标回收率85%-115%" |
| 统计方法不清 | "进行统计分析" | "采用SPSS 26.0，Mann-Whitney U检验（非正态数据）" |
| 参数不完整 | "采用PCA分析" | "采用主成分分析(PCA)，以特征值>1为提取标准" |
| 缺少采样信息 | "采集了水样" | "2024年1-4月，在9个监测点每月采集1次" |

## 五、结果（Results）常见错误

| 错误 | 说明 |
|------|------|
| 结果与讨论混杂 | 结果段落中插入解释和文献对比 |
| 图表重复 | 同一数据既画图又列表 |
| 缺少统计检验 | "A比B高"没有p值 |
| 数字不一致 | 正文数字与表格数字不匹配 |
| 有效数字不当 | "98.4321 mg/L"过度精确 |
| 忽略负结果 | 只报告显著的，不报告不显著的 |

### 数字一致性检查

```
检查项：
1. 正文中每个数字是否与表格对应
2. 百分比加起来是否等于100%
3. 均值±SD的SD是否合理（不应大于均值）
4. 小数位数是否统一
5. 单位是否统一
```

## 六、讨论（Discussion）常见错误

### 6.1 最致命的错误：没有机制解释

```
错误写法（空洞）：
"本研究发现餐饮区TOC浓度显著高于教学区。这一结果与张三[1]
的研究一致。"

正确写法（有机制）：
"本研究发现餐饮区TOC浓度(185.6±42.3 mg/L)显著高于教学区
(98.4±28.7 mg/L)(p<0.05)。这一差异可以从有机物来源角度解释：
餐饮区排放废水含有大量油脂和食物残渣，其有机负荷远高于教学区
实验室废水[1]。Guisasola等[2]研究表明，餐饮废水的BOD/COD比值
通常在0.5-0.7之间，可生化性较强，有利于微生物快速利用。"
```

### 6.2 Discussion结构问题

| 错误 | 说明 |
|------|------|
| 复述结果 | "结果表明A为X，B为Y" → 应解释为什么 |
| 缺少文献对比 | 没有与前人研究的比较 |
| 缺少局限性 | 没有坦诚指出研究局限 |
| 过度推测 | 证据不足就下结论 |
| 与引言重复 | 重复引言中已说的背景 |

### 6.3 常见逻辑漏洞

```
错误：相关性=因果性
"DO降低导致CH4升高" → "DO降低与CH4升高显著相关，可能是
因为..."

错误：以偏概全
"本研究结果表明所有污水管网都..." → "本研究结果表明该校园
污水管网..."

错误：忽视替代解释
只给一种解释，不考虑其他可能原因
```

## 七、结论（Conclusion）常见错误

| 错误 | 说明 |
|------|------|
| 重复摘要 | 结论与摘要内容相同 |
| 过度夸大 | "首次发现" "重大突破" |
| 包含新信息 | 结论中出现了前文未提到的内容 |
| 包含参考文献 | 结论中有"[X]" |
| 缺少定量 | "浓度较高" → "浓度为X mg/L" |

## 八、图表常见错误

| 错误 | 说明 |
|------|------|
| 坐标轴缺单位 | "TOC" → "TOC (mg/L)" |
| 分辨率不足 | 截图直接粘贴 |
| 颜色不友好 | 红绿对比（色盲不可辨） |
| 图例遮挡数据 | 图例位置不当 |
| 三维图 | 二维数据用三维图（不推荐） |
| 缺少子图标号 | 无(a)(b)(c)标注 |
| 表格格式错 | 三线表有多余线条 |

---

## 九、参考文献常见错误

| 错误 | 说明 |
|------|------|
| 格式不统一 | 同一篇论文混用不同格式 |
| 期刊名缩写不一致 | 有的全称有的缩写 |
| 缺少页码 | 无起止页码 |
| 作者格式错 | 姓在前名在后混淆 |
| 自动编号出错 | 编号与正文引用不对应 |
| 包含未引用的文献 | 参考文献列表中有正文未引用的 |

## 十、投稿前检查清单

### 10.1 内容检查

```
□ 标题准确反映研究内容，无缩写
□ 摘要包含目的/方法/结果/结论四要素
□ 摘要中无参考文献编号
□ 关键词覆盖核心概念
□ 引言逻辑链完整（背景→问题→空白→目标）
□ 方法部分有国家标准/文献来源
□ 方法部分有质控措施
□ 结果有统计检验支撑（p值）
□ 结果中数字与表格一致
□ Discussion每个发现有机制解释
□ Discussion有文献对比
□ Discussion有局限性讨论
□ 结论不重复摘要
□ 结论无参考文献
□ 所有图表都有标题和注释
□ 坐标轴都有单位
□ 参考文献格式统一
□ 正文引用与参考文献列表对应
```

### 10.2 格式检查

```
□ 字体字号符合期刊/学位论文要求
□ 行距符合要求
□ 页边距符合要求
□ 页眉页脚正确
□ 页码连续
□ 图片分辨率≥300 dpi
□ 表格为三线表
□ 参考文献格式统一（GB/T 7714或期刊格式）
□ 中英文摘要/关键词对照
□ 作者信息完整
```

### 10.3 语言检查

```
□ 无错别字
□ 标点使用规范（中文全角，英文半角）
□ 单位格式正确（数值+空格+单位）
□ 数字格式统一
□ 无口语化表达
□ 无过度使用"了""的"
□ 英文摘要无语法错误
□ 术语使用统一
```
