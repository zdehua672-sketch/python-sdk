# 图表设计库

## 一、SCI图表规范

### 1.1 图片基本要求

| 要素 | 要求 |
|------|------|
| 分辨率 | ≥300 dpi（线图≥600 dpi） |
| 格式 | TIFF/EPS/PDF（矢量优先） |
| 字体 | Arial/Helvetica/Times New Roman |
| 字号 | 坐标轴标签8-10pt，图例7-9pt |
| 线宽 | 坐标轴≥1pt，数据线≥0.75pt |
| 颜色 | CMYK兼容，色盲友好 |
| 尺寸 | 单栏8.5cm，双栏17.4cm |

### 1.2 表格基本要求

| 要素 | 要求 |
|------|------|
| 格式 | 三线表（无竖线、无横格线） |
| 字体 | 全表统一，通常8-10pt |
| 标题 | 置于表格上方 |
| 注释 | 置于表格下方，用a,b,c标注 |
| 有效数字 | 与测量精度一致，通常保留1-2位小数 |

### 1.3 三线表模板

```
表1  不同功能区碳污染物浓度统计结果
┌──────────────────────────────────────────────────────────────┐
│ 采样区域    TOC (mg/L)    COD (mg/L)    CH4 (mg/L)   CO2 (mg/L) │
├──────────────────────────────────────────────────────────────┤
│ 教学区      98.4±28.7a   156.3±45.2a   0.8±0.3a    12.5±3.8a   │
│ 生活区      112.5±31.2ab  189.6±52.4ab  1.2±0.5ab   15.3±4.2ab  │
│ 餐饮区      185.6±42.3b  312.8±68.5b   2.1±0.8b    22.7±5.6b   │
└──────────────────────────────────────────────────────────────┘
注：同列不同字母表示差异显著(p<0.05)
```

## 二、各类图表适用场景

### 2.1 柱状图/条形图
- **适用：** 不同组别的均值比较（≤6组）
- **不适用：** 连续变量趋势、大量分组
- **改进：** 叠加散点图展示个体值

### 2.2 箱线图
- **适用：** 数据分布对比、异常值检测
- **优势：** 展示中位数、四分位距、极值
- **本项目：** 不同功能区TOC/COD/CH4分布对比

### 2.3 热力图
- **适用：** 相关性矩阵、时空分布
- **颜色方案：** 相关性用RdBu_r（红蓝），浓度用YlOrRd
- **本项目：** 9个指标×多个采样点的浓度热力图

### 2.4 PCA双标图
- **适用：** 降维可视化、变量关系
- **要素：** 样本点+载荷箭头+方差贡献率
- **本项目：** 碳污染物多指标PCA分析

### 2.5 树状图（Dendrogram）
- **适用：** 聚类分析结果可视化
- **要素：** 距离标尺+最优聚类数标注
- **本项目：** 采样点按碳污染物特征的HCA聚类

### 2.6 散点图+回归线
- **适用：** 两变量关系、回归模型
- **要素：** 散点+拟合线+置信区间+R²+p值
- **本项目：** DO-CH4关系、TOC-CH4关系

### 2.7 折线图
- **适用：** 时间序列、沿程变化趋势
- **本项目：** 管网沿程DO/CH4/CO2变化

### 2.8 堆叠柱状图
- **适用：** 组成比例对比
- **本项目：** DOC/POC组成对比

## 四、图表标题写法

### 4.1 中文标题格式

```
图1  不同功能区TOC浓度分布（a：教学区；b：生活区；c：餐饮区）
图2  污水管网沿程DO、CH4、CO2浓度变化趋势
图3  碳污染物指标主成分分析双标图（PC1解释方差52.3%，PC2解释方差21.7%）
图4  采样点层次聚类分析树状图（Ward法，欧氏距离）
图5  DO浓度与CH4浓度的回归分析（灰色区域为95%置信区间）
```

### 4.2 英文标题格式（SCI论文）

```
Fig. 1  Spatial distribution of TOC concentrations across functional zones
       (a: Teaching area; b: Residential area; c: Food service area)
Fig. 2  Longitudinal variation of DO, CH4, and CO2 concentrations
       along the sewage pipeline
Fig. 3  PCA biplot of carbon pollutant indicators
       (PC1 explained 52.3% variance, PC2 explained 21.7%)
```

### 4.3 表格标题格式

```
表1  不同功能区碳污染物浓度统计结果（Mean±SD）
Table 1  Statistical results of carbon pollutant concentrations
        across functional zones (Mean±SD)
```

## 五、图表注释规范

### 5.1 统计标注

```
*   p < 0.05
**  p < 0.01
*** p < 0.001
ns  不显著

字母标注法：同列不同字母表示差异显著(p<0.05)
```

### 5.2 图注信息

每张图的Figure Caption应包含：
1. 图的核心发现（一句话）
2. 统计方法说明
3. 符号/颜色含义解释
4. 样本量（n=X）

**模板：**
```
图[X] [图表类型]展示[内容]。采用[统计方法]，n=[样本量]。
[符号说明]。[核心发现]。
```

## 六、多图排版原则

### 6.1 常用布局

```
单图：     2×2网格：      1×3横排：     2×2+底部单图：
┌──────┐   ┌────┬────┐   ┌───┬───┬───┐   ┌────┬────┐
│  (a) │   │(a) │(b) │   │(a)│(b)│(c)│   │(a) │(b) │
│      │   ├────┼────┤   └───┴───┴───┘   ├────┼────┤
│      │   │(c) │(d) │                    │(c) │(d) │
└──────┘   └────┴────┘                    ├──────────┤
                                          │   (e)    │
                                          └──────────┘
```

### 6.2 子图标注

- 左上角标注(a)(b)(c)(d)
- 字号比图内文字大2pt
- 统一使用Arial Bold

### 6.3 排版建议

- 子图间距：水平0.5cm，垂直0.5cm
- 图例位置：优先放在图内空白处
- 共用图例：当子图含义相同时，使用共用图例

## 七、本项目图表优化建议

### 7.1 已实现图表（plotting_functions.py）

| 图表类型 | 方法 | 本项目用途 |
|---------|------|-----------|
| 饼图 | pie_chart() | 样品类型/相态组成 |
| 箱线图 | boxplot() | 功能区浓度分布 |
| 热力图 | heatmap() | 指标相关性矩阵 |
| PCA双标图 | pca_biplot() | 碳污染物降维分析 |
| 树状图 | dendrogram() | 采样点聚类 |
| 回归图 | regression_plot() | DO-CH4关系 |
| 柱状图 | bar_chart() | 功能区均值对比 |
| 分组柱状图 | grouped_bar() | 冬季vs春季对比 |
| 显著性标注柱状图 | bar_with_significance() | 统计差异展示 |

### 7.2 本项目建议图表组合

**硕士论文：**
```
图1: 采样点分布图（GIS地图）
图2: 不同功能区TOC/COD箱线图（3×1）
图3: 冬季vs春季浓度对比分组柱状图
图4: 沿程DO/CH4/CO2变化折线图（双Y轴）
图5: 9指标相关性热力图
图6: PCA双标图
图7: HCA树状图
图8: DO-CH4回归图（含95%CI）
```

**SCI论文（精选6个）：**
```
Fig.1: TOC/COD箱线图（3 zones × 2 seasons）
Fig.2: 沿程变化趋势
Fig.3: 相关性热力图
Fig.4: PCA biplot
Fig.5: Regression with CI
Fig.6: Conceptual diagram（碳平衡示意图）
```

### 7.3 常见图表错误

- 颜色过多（>7种颜色）
- 坐标轴标签不带单位
- 图例遮挡数据
- 分辨率不足（<300 dpi）
- 字号过小（<7pt）
- 缺少误差棒
- 三维图（一般不推荐）
